# BigQuery Ecommerce Analysis

## About
SQL portfolio project analysing Google Analytics ecommerce data using BigQuery and Python.

## Background
## Background
Experienced financial analyst with advanced Excel skills, expanding into SQL and Python for data analysis.

## Project
This project is based on the Google Cloud Skills Boost lab: **"Exploring an Ecommerce Dataset using SQL in Google BigQuery"**, extended with additional analysis in a Jupyter notebook.

## Dataset
Public Google Analytics dataset: `data-to-insights.ecommerce.all_sessions`
- 21.5 million rows of ecommerce session data from the Google Merchandise Store

## What I analysed
- Data quality check — identifying and removing duplicate records
- Unique visitors by channel (Organic Search, Direct, Referral, etc.)
- Top 5 most viewed products
- Product conversion rates
- Checkout funnel analysis
- Abandoned cart detection from high quality sessions

## Tools
- BigQuery (SQL)
- Python (pandas, google-cloud-bigquery)
- Jupyter Notebook
